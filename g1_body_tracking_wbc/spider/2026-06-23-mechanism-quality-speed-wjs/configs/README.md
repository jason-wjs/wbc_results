# Configs

- `g1_wbc_reward_weights_method_specific_v14_20260612.json`: reward weights
  used by the mechanism sweeps.
- `source_design.md`: design context copied from the source repo.
- `source_plan.md`: execution plan copied from the source repo.
- `summaries/*.csv`: the 11 copied summary CSV files used to build the package
  report, `primary_metrics.csv`, and `run_index.csv`.

Raw `metrics.json`, `rollout.npz`, `mpc_command.npz`, videos, and logs are not
stored here. They are indexed in `artifacts.md`.
